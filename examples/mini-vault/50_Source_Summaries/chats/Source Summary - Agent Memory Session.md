# Source Summary - Agent Memory Session

## Summary

The session defines a safe memory workflow for agents: preserve raw evidence,
write summaries as interpretation, route uncertain claims to review, and avoid
silently promoting guesses into durable concepts.

## Evidence Notes

- Raw transcripts remain evidence.
- Interpretation belongs in summaries, concepts, decisions, or review queues.
- Uncertain claims should be labeled and reviewed before becoming durable memory.
